What is YMLA?

  • YQFC is a web tool to compare 85 quantitative features between two yeast gene lists.

  • For each quantitative feature, YQFC provides three statistical tests (t-test, U test, and KS test) to test whether this quantitative feature is significantly different between the two yeast gene lists.

  • Both tables and figures (box plots and cumulative distribution function plots) are provided to visualize the testing results.
